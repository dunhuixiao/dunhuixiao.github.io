    IDEA Setting Repository 同步仓库报错解决
    最近更新：2025-11-21   |   字数总计：92   |   阅读估时：1分钟   |   阅读量：次
    1. 报错 Commit on repo without HEAD currently not supported
      1. 将当前设置导出备份，然后删除 C:\Users\USERNAME\AppData\Roaming\JetBrains\IntelliJIdeaXXX\settingsRepository
      2. reset IDEA 后，再次尝试同步，正常
    2. 报错 git upload pack
      1. 网络问题，换成 gitee 源
      2. 在 IDEA 内配置科学上网 2. 正常电脑的科学上网链接成功后，将 IDEA Proxy 内的设置修改为 auto 即可
